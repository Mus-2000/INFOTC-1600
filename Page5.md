**Exporting audio in Logic Pro X**


After recording and adding effects to your audio recording, the final step is to export it.

Navigate to the file tab at the top


<img width="1440" alt="Screen Shot 2021-12-01 at 8 07 56 PM" src="https://user-images.githubusercontent.com/54383601/144344486-76ad727c-73d8-439f-a255-aac124c681ad.png">


scroll all the way down to bounce and then click on project or section

<img width="1440" alt="Screen Shot 2021-12-01 at 8 08 10 PM" src="https://user-images.githubusercontent.com/54383601/144344544-628e743f-7088-4a6b-9232-a399122f5a4d.png">

this is what the screen should look like

<img width="1440" alt="Screen Shot 2021-12-01 at 8 08 21 PM" src="https://user-images.githubusercontent.com/54383601/144344595-bf5a097f-3fae-4e61-b09a-da86cb5110b7.png">

Select MP3 if you want an MP3 file and select PCM if you want a wav file and then select OK

The file will populate in the location you selected.
